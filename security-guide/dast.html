Dynamic Scanning (DAST)

Finding vulnerabilities in a running application with Nuclei and Invicti.

What is DAST?

Dynamic Application Security Testing (DAST) tests the application from the outside, just as an attacker would. It interacts with the running application to find vulnerabilities like SQL Injection, XSS, and broken authentication.

Tool 1: Nuclei

Nuclei is a fast, template-based vulnerability scanner. It allows us to scan for thousands of known CVEs and misconfigurations using community-powered YAML templates.

# Run a quick local scan
./run-nuclei.sh

Tool 2: Invicti

Invicti (formerly Netsparker) provides enterprise-grade, depth-first DAST scanning. It uses a "Proof-Based Scanning" technology to verify vulnerabilities and reduce false positives.

  • Local Scan Agent for secure environments.
  • Comprehensive coverage of complex web apps.
  • Automated verification of findings.

Local Agent Strategy

Since our application runs in a local or staging environment, we use an Invicti Scan Agent. This agent acts as a bridge, allowing the cloud-based Invicti platform to safely scan your private local infrastructure without exposing it to the public internet.
